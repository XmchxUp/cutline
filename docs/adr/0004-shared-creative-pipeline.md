# Creative workflows share one pipeline

Cutline will model FilmCommentaryEdit and StoryHighlightVideo as variants of a shared CreativePipeline instead of building separate end-to-end flows for each workflow. The workflows differ in how they analyze sources and assemble visuals, but sharing ingestion, selection, scripting, draft packaging, preview, review, and render concepts keeps draft packages consistent and avoids duplicating product behavior as new creative workflows are added.
